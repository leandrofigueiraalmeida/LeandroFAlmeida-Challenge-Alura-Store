<h1 align="center">📊 Análise de Desempenho – Alura Store</h1>

<h2>📌 Objetivo do Projeto</h2>
<p>
Durante este desafio, o objetivo foi ajudar o <strong>Senhor João</strong> a decidir qual loja da sua rede <strong>Alura Store</strong> deve ser vendida para viabilizar um novo empreendimento.
</p>

<p>
A análise foi baseada em dados extraídos de arquivos CSV contendo informações de vendas, desempenho e avaliações das quatro lojas da rede.
</p>

<p>
O foco principal foi identificar <strong>a loja com menor eficiência operacional e comercial</strong>, utilizando métricas quantitativas e visualizações de dados.
</p>

<hr>

<h2>🛠️ Tecnologias Utilizadas</h2>
<ul>
  <li>🐍 Python</li>
  <li>📦 Pandas</li>
  <li>📊 Matplotlib</li>
  <li>📓 Jupyter Notebook</li>
</ul>

<hr>

<h2>📂 Estrutura da Análise</h2>

<h3>📦 1. Carregamento dos Dados</h3>
<ul>
  <li>Importação dos arquivos CSV via URLs do repositório da Alura</li>
  <li>Organização dos dados em DataFrames utilizando <strong>Pandas</strong></li>
</ul>

<h3>🧮 2. Cálculo das Métricas</h3>
<ul>
  <li>💰 Faturamento total por loja</li>
  <li>⭐ Avaliação média dos clientes</li>
  <li>🚚 Frete médio</li>
  <li>🛍️ Categorias mais vendidas</li>
  <li>🔝 Produtos mais vendidos</li>
  <li>📈 Pontuação composta (indicador geral de desempenho)</li>
</ul>

<hr>

<h2>📊 Visualizações Criadas</h2>

<p>Foram desenvolvidos <strong>5 gráficos diferentes utilizando Matplotlib</strong>, incluindo:</p>

<ul>
  <li>📊 Gráfico de barras – Faturamento por loja</li>
  <li>📊 Barras horizontais – Categorias mais vendidas</li>
  <li>🥧 Gráfico de pizza – Participação no faturamento total</li>
  <li>📈 Gráfico de dispersão – Relação entre avaliação e faturamento</li>
  <li>📊 Comparativo de categorias por loja</li>
</ul>

<p>
As visualizações ajudam a interpretar os dados de forma clara e facilitar a tomada de decisão estratégica.
</p>

<hr>

<h2>📈 Principais Análises Realizadas</h2>
<ul>
  <li>Comparação de desempenho financeiro entre as lojas</li>
  <li>Avaliação da satisfação dos clientes</li>
  <li>Análise de eficiência logística (frete médio)</li>
  <li>Identificação de padrões de venda por categoria</li>
  <li>Construção de uma métrica composta para decisão final</li>
</ul>

<hr>

<h2>📝 Recomendação Final</h2>

<p>
Com base na análise dos dados, foi identificada a loja com menor desempenho considerando:
</p>

<ul>
  <li>Menor faturamento</li>
  <li>Avaliação média inferior</li>
  <li>Frete menos competitivo</li>
  <li>Baixa performance relativa na pontuação composta</li>
</ul>

<p>
<strong>📌 Recomendação:</strong> Vender a loja com menor eficiência operacional, pois apresenta menor retorno estratégico comparado às demais unidades.
</p>

<p>
A decisão foi baseada exclusivamente em dados quantitativos e análise comparativa entre as quatro lojas.
</p>
